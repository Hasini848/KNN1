KNN works using similarity and distance.It doesn't learn using the equations like logistic,linear and svm

How KNN works?
Step1:Choose the K-Value
Step2:Find the nearest neighbors
Step3:Check the majority class
Step4:Assign predicted class

Distance Metrics:Knn depends on distance
 1. Eucldiean Distance= Continuous numerical data
 sqrt((x2-x1)**2 + (y2-y1)**2)
 2. Manhattan Distance = Grid based distance
 3. Minkwski = Generalized distance
 4. Hamming =  Binary Categorical Data

 Mostly odd k values are preferred

 KNN can also be used for classification as well as the regression
